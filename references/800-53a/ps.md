# NIST SP 800-53A Rev 5 — PS: Personnel Security Assessment Procedures

Assessment procedures for the Personnel Security family evaluate whether the organization screens individuals, manages terminations and transfers, and maintains personnel security controls.

---

## PS-1: Policy and Procedures

**Assessment Objective:** Determine if the organization develops, documents, disseminates, reviews, and updates personnel security policy and procedures.

**Examine:** Personnel security policy; personnel security procedures; system security plan.

**Interview:** Organizational personnel responsible for personnel security policy.

**Test:** Organizational processes for managing personnel security policy.

**Determination Statements:**
- PS-01a. — A personnel security policy is developed, documented, and disseminated
- PS-01b. — Personnel security procedures are developed, documented, and disseminated
- PS-01c. — The policy and procedures are reviewed and updated at an organization-defined frequency

---

## PS-2: Position Risk Designation

**Assessment Objective:** Determine if the organization assigns risk designations to all organizational positions and establishes screening criteria.

**Examine:** Personnel security policy; position risk designation records; position descriptions; screening criteria documentation.

**Interview:** Human resources personnel; organizational personnel responsible for position risk designations.

**Test:** Position risk designation processes.

**Determination Statements:**
- PS-02a. — A risk designation is assigned to all organizational positions
- PS-02b. — Screening criteria for individuals filling those positions are established
- PS-02c. — Position risk designations are reviewed and updated at an organization-defined frequency

---

## PS-3: Personnel Screening

**Assessment Objective:** Determine if the organization screens individuals prior to authorizing access to the system.

**Examine:** Personnel security policy; personnel screening procedures; screening records; system security plan; records of rescreening.

**Interview:** Human resources personnel; organizational personnel responsible for personnel screening.

**Test:** Personnel screening processes; rescreening processes.

**Determination Statements:**
- PS-03a. — Individuals are screened prior to authorizing access to the system
- PS-03b. — Individuals are rescreened in accordance with organization-defined conditions requiring rescreening and at an organization-defined frequency

---

## PS-4: Personnel Termination

**Assessment Objective:** Determine if the organization manages personnel termination with respect to information system access.

**Examine:** Personnel security policy; personnel termination procedures; records of terminated employees; system access revocation records; exit interview records; property return records.

**Interview:** Human resources personnel; organizational personnel responsible for personnel termination; system administrators.

**Test:** System access disabling mechanisms upon termination; authenticator revocation processes.

**Determination Statements:**
- PS-04a. — System access is disabled within an organization-defined time period upon termination of individual employment
- PS-04b. — Authenticators and credentials associated with the individual are terminated/revoked
- PS-04c. — Exit interviews are conducted that include a discussion of organization-defined information security topics
- PS-04d. — All security-related organizational system-related property is retrieved
- PS-04e. — Access to organizational information and systems formerly controlled by the terminated individual is retained
- PS-04f. — Organization-defined personnel or roles are notified within an organization-defined time period

---

## PS-5: Personnel Transfer

**Assessment Objective:** Determine if the organization reviews and confirms ongoing operational need for current access when individuals are reassigned or transferred.

**Examine:** Personnel security policy; personnel transfer procedures; access authorization records; system access records.

**Interview:** Human resources personnel; organizational personnel responsible for personnel transfers; system administrators.

**Test:** Access authorization review processes upon transfer; access modification mechanisms.

**Determination Statements:**
- PS-05a. — Ongoing operational need for current logical and physical access authorizations is reviewed and confirmed when individuals are reassigned or transferred
- PS-05b. — Organization-defined transfer or reassignment actions are initiated within an organization-defined time period
- PS-05c. — Access authorization is modified as needed to correspond with changes in operational need
- PS-05d. — Organization-defined personnel or roles are notified within an organization-defined time period

---

## PS-6: Access Agreements

**Assessment Objective:** Determine if the organization develops, documents, and manages access agreements.

**Examine:** Personnel security policy; access agreement documentation; signed access agreements; system security plan.

**Interview:** Organizational personnel responsible for access agreements; system users.

**Test:** Access agreement review and update processes; re-signing processes.

**Determination Statements:**
- PS-06a. — Access agreements for organizational systems are developed and documented
- PS-06b. — Access agreements are reviewed and updated at an organization-defined frequency
- PS-06c. — Individuals requiring access to organizational information and systems sign appropriate access agreements prior to being granted access and re-sign when access agreements have been updated or at an organization-defined frequency

---

## PS-7: External Personnel Security

**Assessment Objective:** Determine if the organization establishes personnel security requirements for external providers.

**Examine:** Personnel security policy; external personnel security procedures; contracts with external providers; personnel security requirement documentation; compliance monitoring records.

**Interview:** Organizational personnel responsible for external personnel security; contracting officers.

**Test:** External personnel security compliance monitoring mechanisms.

**Determination Statements:**
- PS-07a. — Personnel security requirements are established including security roles and responsibilities for external providers
- PS-07b. — External providers comply with personnel security policies and procedures established by the organization
- PS-07c. — Personnel security requirements are documented
- PS-07d. — External providers notify organization-defined personnel of personnel transfers or terminations within an organization-defined time period
- PS-07e. — Provider compliance with personnel security requirements is monitored

---

## PS-8: Personnel Sanctions

**Assessment Objective:** Determine if the organization employs a formal sanctions process for individuals failing to comply with security policies.

**Examine:** Personnel security policy; personnel sanctions procedures; records of formal sanctions; system security plan.

**Interview:** Human resources personnel; organizational personnel responsible for personnel sanctions; information system security officers.

**Test:** Sanctions notification processes.

**Determination Statements:**
- PS-08a. — A formal sanctions process is employed for individuals failing to comply with established information security and privacy policies and procedures
- PS-08b. — Organization-defined personnel or roles are notified within an organization-defined time period when a formal employee sanctions process is initiated
